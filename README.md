# sp-copilot-search-api-component
> Copilot Search API Component is an SPFx extension that has been built with SPFx 1.22. The SPFx extension show case the power of Copilot Search API in SharePoint where we could perform hybrid search  across OneDrive using natural language queries that understand context and intent to return relevant documents and files.
